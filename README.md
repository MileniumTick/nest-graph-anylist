# Dev Enviroment

1. Clonar el proyecto
2. Copiar el ```env.template``` y renombrar a ```.env```
3. Ejecutar

  ```bash
  pnpm install
  ```

4. Levantar la imagen (Docker)

  ```bash
  docker compose up -d
  ```

5. Levantar el backend de nest

  ```bash
  pnpm start:dev
  ```
